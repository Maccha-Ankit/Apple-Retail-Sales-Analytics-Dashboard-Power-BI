#  Apple Sales Analytics Dashboard | Power BI

## 📌 Project Overview

This project presents an interactive Power BI dashboard built to analyze over **1 million+ Apple retail sales records**. The dashboard provides insights into sales performance, profitability, product trends, regional performance, and month-over-month growth through dynamic visualizations and KPIs.

The objective of this project is to demonstrate practical data analytics skills including data cleaning, data modeling, DAX calculations, and dashboard design using Power BI.

---

## 🎯 Objectives

* Analyze Apple retail sales performance.
* Identify top-performing products and categories.
* Monitor revenue and profit trends over time.
* Compare regional sales performance.
* Track Month-on-Month (MoM) growth.
* Build an interactive dashboard for business decision-making.
